## 第六天

- 文本对象
  语法：`operator + （内部/外部） + 文本对象`
  可视化模式 + （内部/外部） + 文本对象
  内部：`i`
  外部：`a`

- 文本对象：

  一个单词 ： `w`

  一对() : `(`或`)`或 `b`

  一对[] : `[`或`]`

  一对{} : `{`或`}`或 `B`

  一对<> : `<`或`>`

  XML 标签 : `t`

  一对' : `'`

  一对" : `"`

  一对 \` : `` ` ``

  一个句子 : `s`

  一个段落 : `p` 指换行

- vscode 的 vim 默认集成的插件

  - vim-textobj-arguments : `a`

  `ia` 不包含分隔符

  `aa` 包含分隔符

  技巧：删除一个参数：`daa`

  修改一个参数：`cia`

  - vim-textobj-entire : `e`

  `ie` 删除当前文本所有内容但是不包含前后的空格

  `ae` 删除当前文本所有内容
